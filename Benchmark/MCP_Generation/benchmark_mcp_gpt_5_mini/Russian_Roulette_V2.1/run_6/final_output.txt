Summary of the main steps I used to construct the game:
1. Created the game with players ["Player 1", "Player 2"].
2. Added a chance move at the root selecting the loaded chamber {1,..,6} with probabilities 1/6 each.
3. For each chamber index k = 1..6 I created an infoset for the player whose turn it is at that chamber, grouping all paths corresponding to unknown loaded chambers consistent with reaching chamber k. Each infoset has actions ["Quit","Pull"].
4. For each path I set terminal outcomes:
   - If the active player chooses Quit: quitter gets 0, other gets 1.
   - If the active player Pulls and the loaded chamber equals the current chamber: the pulling player dies (-1), other player wins (1).
   - If the active player Pulls and the loaded chamber is later: the game continues to the next chamber (handled by the next infoset).
5. Validated the complete game (all terminal nodes have outcomes) and saved the game file as russian_roulette.efg in the current working directory.
